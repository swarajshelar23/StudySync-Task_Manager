# StudySync Frontend (Vite + React)

Quick frontend to interact with StudySync Spring Boot API.

Run locally:

1. cd frontend
2. npm install
3. npm run dev

Default API base: `http://localhost:8080/api/tasks` (change `REACT_APP_API_BASE` env var to override).

Git branch strategy (Git Lab 3):

- Create feature branch: `feature/frontend`
- Commit regularly while developing
- Open PR targeting `develop` branch when ready
